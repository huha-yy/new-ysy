## 📝 下载步骤

1. **点击上面的图片链接**
2. **在浏览器中右键点击图片**
3. **选择"图片另存为..."**
4. **保存到：**`frontend/public/images/activities/`目录
5. **重命名为对应的文件名**

---

## 💡 提示

- 图片尺寸：建议 800x450（宽高比约 1.78:1）
- 文件格式：JPG 或 PNG
- 如果不想下载，也可以暂时使用占位图片（后续替换）
