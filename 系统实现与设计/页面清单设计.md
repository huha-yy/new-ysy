# 户外徒步活动管理系统 - 页面清单设计

## 一、页面规划概述

### 1.1 访问角色

| 角色 | 可访问页面范围 |
|------|----------------|
| 游客（未登录） | 公共页面（首页、活动列表、活动详情、登录、注册） |
| 参与者 | 公共页面 + 个人中心 + 报名 + 签到 + 评价 |
| 组织者 | 参与者页面 + 活动管理 + 报名审核 |
| 管理员 | 所有页面 + 管理后台 |

### 1.2 页面布局

| 布局类型 | 适用页面 | 说明 |
|----------|----------|------|
| 空白布局 | 登录、注册 | 无导航栏 |
| 主布局 | 前台所有页面 | 顶部导航 + 内容区 |
| 管理布局 | 后台管理页面 | 侧边栏 + 顶栏 + 内容区 |
